# to_do_list_api

```
Ruby 3.4.4
Rails 8.0.3
PostgreSQL 16.8
```

Comandos para construir o projeto: 
```
 bundle install
 bin/rails db:create db:migrate db:seed
 bin/rails s
```
http://localhost:3000

Rodar tests
```
bundle exec rspec
```
